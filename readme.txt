Open project with preferably Unity version: 2019.4.18f1

Project should build and run after pressing Play button in the editor.

--------------Controls--------------
On main menu:
Spacebar to start game.
Left click on 'START' with mouse to start game.

During game:
Arrow and WASD keys to move snake. 

On Game Over screen:
Left click on 'Play Again' with mouse to restart game.

Press Esc to exit game anytime.
